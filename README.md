# Siberia Price

Исследование удельной кадастровой стоимости промышленных земель в Республике Тыва и регионах Сибири.

## Цель проекта

Проверить гипотезу о возможном завышении кадастровой стоимости земельных участков промышленного назначения в Республике Тыва относительно сопоставимых индустриальных локаций Сибири.

Проект должен сформировать воспроизводимую основу для сбора данных, нормализации видов разрешённого использования (ВРИ), расчёта метрик и сравнительного анализа регионов.

## Источники данных

- НСПД и сведения Росреестра / ЕГРН;
- Фонд данных государственной кадастровой оценки;
- сведения о сделках ЕГРН;
- объявления о продаже и банкротные торги;
- OpenStreetMap и другие открытые геоданные;
- официальная экономическая статистика регионов.

Условия доступа, ограничения и назначение источников описаны в [`docs/data_sources.md`](docs/data_sources.md). Массовый сбор данных из НСПД пока не реализован: клиент остаётся каркасом до фиксации актуальных endpoint, лимитов и правил хранения исходных данных.

## Базовая логика

1. Собрать массив промышленных земельных участков.
2. Нормализовать ВРИ.
3. Рассчитать кадастровую стоимость 1 м².
4. Классифицировать локации: городская промзона, пригородная индустриальная зона, периферия.
5. Сравнить Туву с регионами Сибири.
6. Проверить кадастровую стоимость через рынок и налоговую нагрузку.
7. Подготовить доказательную базу для управленческого разговора.

## Структура

```text
docs/        методология и проектная документация
config/      регионы, ВРИ, правила очистки
data/        raw/interim/processed/external данные
src/siberia_price/  устанавливаемый Python-пакет
outputs/     таблицы, карты, графики, отчеты
app/         будущий Streamlit-дашборд
tests/       тесты
```

## Быстрый старт

```bash
python -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
python -m pip install -e .
python -m pytest
```

В Windows PowerShell команда активации отличается:

```powershell
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
python -m pip install -e .

```
