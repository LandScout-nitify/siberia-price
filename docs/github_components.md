# Open-source компоненты и технический стек

## Кадастровые данные

### rosreestr2coord

Референс для получения координат и геометрии участка по кадастровому номеру через НСПД.

Использование в проекте: изучить подход, но основную логику желательно вынести в собственный модуль `src/siberia_price/nspd`.

### mcp-rosreestr

MCP-обертка для работы с Росреестром/НСПД.

Использование в проекте: опционально, если потребуется агентный интерфейс.

## Геоаналитика

- GeoPandas;
- Shapely;
- pyproj;
- OSMnx;
- pyrosm;
- H3 — опционально.

## Хранение данных

На старте:

- Parquet / GeoParquet;
- DuckDB.

PostGIS — позже, если проект станет сервисом.

## Нормализация ВРИ

- собственный словарь ВРИ;
- rapidfuzz для полуавтоматической классификации.

## Визуализация

- Folium;
- Plotly;
- Streamlit;
- QGIS для ручной проверки.

## Принцип

Сырые данные сохраняются отдельно от обработанных. Любой расчет должен быть воспроизводимым.
