# Задача для Codex: первый коммит проекта siberia-price

## Контекст

Проект `siberia-price` предназначен для исследования удельной кадастровой стоимости промышленных земель в Республике Тыва и регионах Сибири.

Главная гипотеза: кадастровая стоимость промышленных земель в Туве может быть завышена относительно сопоставимых индустриальных локаций Сибири, что создает налоговую и инвестиционную нагрузку для бизнеса.

## Что нужно сделать

Создай стартовую структуру проекта согласно файлам из этого архива.

## Требования

1. Сохрани структуру каталогов.
2. Не коммить сырые данные.
3. Сохрани `.gitkeep` в пустых папках данных и outputs.
4. Установи зависимости из `requirements.txt`.
5. Проверь, что тесты запускаются.
6. Не реализуй массовый сбор НСПД до отдельной задачи.
7. Оставь `nspd/client.py` как каркас с `NotImplementedError`.
8. Главный упор сейчас — методология, структура данных и воспроизводимый пайплайн.

## После применения

Выполни:

```bash
pip install -r requirements.txt
pip install pytest
pytest
```
